# Security Policy

## Supported versions

The following versions of Astro Landing Kit are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a vulnerability

If you discover a security vulnerability, please report it by emailing
[opensource@lospaccabit.dev](mailto:opensource@lospaccabit.dev).

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Possible impact
- Any suggested remediation

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Disclosure policy

We follow a coordinated disclosure process. Once a fix is released, we will
publish a security advisory and credit the reporter unless they wish to remain
anonymous.
